About this map

Mapped incident sources: a ~90-incident curated layer embedded in this file (works fully offline), plus — when hosted — EuRepoC (2000–2024), VCDB, and recent-months ransomware leak-site claims from Ransomware.live, deduplicated across sources. Incident actors link to a cited threat-group directory (MITRE ATT&CK + Ransomware.live profiles). MITRE ATT&CK and CISA KEV load as enrichment. Origins are attribution or ecosystem proxies — see each incident's note.

The strip above the timeline is an attacks-per-month histogram (2000 → today) — click any bar to jump there. The on-map counter tracks cumulative documented attacks and the tempo multiplier versus the 2000–02 baseline. The green line is a year-on-year trend of peak monthly activity. Press ⟲ Replay to watch the monthly missile salvos escalate.

Methodology

Records are normalised to a common incident schema and deduplicated across sources using date, victim country and a suffix-normalised victim-name key. Ransomware.live records are leak-site claims — attacker assertions, placed at country level, with no origin asserted. Unknown attacker locations launch from a neutral ocean anchor. Incident actors are matched against a threat-group directory built from an embedded core plus MITRE ATT&CK intrusion sets and Ransomware.live group profiles, each cited with links.
